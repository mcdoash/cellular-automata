MATH 4805: Finite Automata final project. Create elementary cellular automata from a Wolfram code and view a visualization of its evolution.

Running

The full demo can be viewed locally via the dist/index.html file without having to install anything. Just open the HTML file in any browser. It can also be viewed online here.

File Structure

├── index.html        # HTML input file
├── src               # Other source code
│   ├── main.js       # UI setup functions
│   ├── automata.mjs  # Cellular automata functions
│   ├── style.scss    # Stylesheet
├── dist              # Final distribution files
│   ├── index.html    # Single-file demo
├── docs              # Code documentation
